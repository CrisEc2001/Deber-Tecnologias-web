Definición

CSS es el lenguaje que controla la presentación y estilo visual de los elementos HTML. Permite definir colores, tipografías, tamaños, márgenes, posiciones y diseños responsivos.

Características principales

  • Lenguaje de cascada: Los estilos se aplican en orden de prioridad.
  • Separación de conceptos: Divide el contenido (HTML) del diseño.
  • Consistencia: Un solo archivo CSS puede controlar todo un sitio web.
  • Adaptabilidad: Uso de Media Queries para diseño responsivo.
